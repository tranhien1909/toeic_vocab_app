# TOEIC Vocab App (Local)

Lightweight local TOEIC vocabulary learning web app.

Requirements
- Node.js (16+)

Quick start

1. Install dependencies

```bash
npm install
```

2. Initialize the SQLite database and import sample data

```bash
npm run init-db
npm run import-data
```

3. Start the server

```bash
npm start
```

4. Open the frontend

Visit http://localhost:3000

Project structure

See the `toeic-vocab-app/` layout in the assignment. Backend runs Express and serves API and static frontend.
